# 小区停车位管理系统 java974

## 项目概述

小区停车位管理系统是一款基于前后端分离架构的信息管理系统，旨在为小区物业管理部门提供便捷的车位管理服务，同时为用户提供在线车位预定、租赁和购买功能。

## 技术栈

- 架构: B/S、MVC
- 开发环境：JDK 1.8、MySQL 5.7以上
- 技术框架：SpringBoot、Vue3、ElementUI、Node.js 14

## 系统角色

系统包括以下三个角色：

- 管理员
- 用户
- 物业管理员

## 功能模块

### 管理员角色

- 用户管理
  - 添加、删除和修改用户信息
- 配置管理
  - 管理系统配置参数
- 权限管理
  - 分配和管理角色权限
- 轮播图管理
  - 配置和显示轮播图
- 物业管理员管理
  - 管理物业管理员信息
- 车位类型管理
  - 添加、删除和修改车位类型
- 出租车位管理
  - 添加、删除和修改出租车位
- 出售车位管理
  - 添加、删除和修改出售车位
- 公告信息管理
  - 管理公告信息

### 用户角色

- 车位预定
  - 填写并提交预定信息
- 车位购买
  - 填写购买信息并支付
- 车位租赁
  - 填写租赁信息并支付
- 临时登记
  - 登记临时停车信息
- 个人信息管理
  - 管理个人信息

### 物业管理员角色

- 车位管理
  - 添加、删除和修改车位信息
- 车位预定审核
  - 审核预定信息
- 车位购买审核
  - 审核购买信息
- 车位租赁审核
  - 审核租赁信息
- 临时登记管理
  - 管理临时停车登记
- 个人信息管理
  - 管理个人信息
